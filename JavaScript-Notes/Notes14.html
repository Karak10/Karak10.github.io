The if, else if and else Statements

Very often when you write code, you want to perform different actions based on different conditions, you can do this by using conditional statements in your code, such as if, else and else if. Before in some of my examples I did use conditional statements, I basically used if, if else and else all together by using symbols to save me time, it's pretty easy actually, see this code for example:

JavaScript Code:

<script> var x = '10'; (x < 5)? document.write('x < 5'): (x === 10)? document.write('x = 10'): document.write('Example'); </script>

Outcome:

This exact same code could also be written like this:

JavaScript Code:

<script> var x = '10'; if (x < 5) { document.write('x < 5'); } else if (x === 10) { document.write('x = 10'); } else { document.write('Example'); } </script>

Outcome:

The switch Statement

In case when you need to test for multiple conditions, writing if elese statements for each condition might not be the best solution. The switch statement is used to perform different actions based on different conditions.

JavaScript Code:

<Script> var day = 2; switch (day) { case 1: document.write("Mondey"); break; case 2: document.write("Tuesday"); break; case 3: document.write("Wednesday"); break; default: document.write("Another day"); } </Script>

Outcome:

After switch keyword you specify the name of the veriety you are checking, then, each case targets a value, if the variety has none of the values you specified then the default option is chosen.

Onced a match has been found all the code following it will also be executed, that's why we use the break keyword, it tells the compiler to stop executing the code in the current block and proceed to the next one. For example, see what happens if I remove all the break keywords from the code I wrote above:

<Script> var day = 2; switch (day) { case 1: document.write("Mondey"); case 2: document.write("Tuesday"); case 3: document.write("Wednesday"); default: document.write("Another day"); } </Script>

Outcome:

Now, let's for example put break after wednesday:

<Script> var day = 2; switch (day) { case 1: document.write("Mondey"); case 2: document.write("Tuesday"); case 3: document.write("Wednesday"); break; default: document.write("Another day"); } </Script>

Outcome:

The For Loop

Loops can execute a block of code a number of times. They are handy in cases in which you want to run the same code repeatedly, adding a different value each time.

JavaScript has three types of loops that you can use: for, while and do while.

The syntax goes like this: for (initialization; condition; afterthoguht) { code block to be executed }

Let's make a simple example, and then I will explain how it works:

JavaScript Code:

<Script> for (x = 1; x<=5; x = x+1) { document.write(x); } </Script>

Outcome:

Alright so, firstly, the easiest to understand, is the first statement, which simply defines what the value of x should be at the beginning, this statement is optional since we could set the value of x before, like this:

JavaScript Code:

<Script> var x = 1; for (; x<=5; x = x+1) { document.write(x); } </Script>

Outcome:

Let's skip the second statement for now and move to the third statement, which defines what should happen for each loop, I wrote x = x+1 so x always increases by one, we could remove this statement too, and write the same thing like this:

JavaScript Code:

<Script> var x = 1; for (; x<=5;) { document.write(x++); } </Script>

Outcome:

The second statement defines when should the loop stop, for example, I wrote x<=5 so the loop stops after it goes beyond 5, if we wanted, we could remove this statement as well, and write it like this:

JavaScript Code:

<Script> var x = 1; for(;;){ if (x<5) break; document.write( x++ ); } </Script>

Outcome:

The While Loop

The while loop repeats through a block of code, as long as a specified condition is true, see the code below for example:

JavaScript Code:

<Script> var x = 1; while ( x < 6 ) { document.write( x++ ); } </Script>

Outcome:

By the way, some people, (me being one of them), when they see this code, they will assume that the final value of x is 5, but that's false, x first becomes 6 and then the system goes like Hauh? x shouldn't be 6 or bigger and stops the loop, if we now wrote document.write(x) for example we would get 6.

JavaScript Code:

<Script> var x = 1; while ( x < 6 ) { x++ } document.write(6); </Script>

Outcome:

Remove the x++ from the code, and the browser will crash, since the loop will be infinite, since x will not increase, there is no way it's ever gonna get past 5.

The Do...While Loop

The do...while loop is a variant of while loop. This loop works basically the same way with while loop, however, it executes the code block once and then checks the condition, which means, even if the condition is false the code block will still be executed once.

JavaScript Code:

<Script> var x = 7; do { document.write( x++ ) } while( x < 6 ); </Script>

Outcome:

Break and Continue

I think I already used break before when I demostrated how to create a for loop without adding any statements, let me now explain how it works.

The break statement is usually used together with if keyword, and it's used to "jump out" of the loop and continue executing the code after the loop, here is an example:

JavaScript Code:

<Script> var x = 1; for (;;x++) { if ( x > 5 ){ break; } document.write( x ) } </Script>

Outcome:

The continue statement breaks only one iteration in the loop, and continues with the next iretation, in the code below for example, number 5 and 6 will be skipped.

JavaScript Code:

<Script> var x = 1; for(; x <= 10; x++){ if( x > 4 && x < 7) { continue; } document.write( x ); } </Script>
