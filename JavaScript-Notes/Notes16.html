JavaScript Arrays

Arrays can store multiple values in a single variable. For example, I could make a variable named colors and give it multiple values, like red, brown, etc, the syntax would look like this:

var colors = new Array("Red", "Brown", "Blue");

You refer to an array element by referring to the index number, written in square bracket, the first value is 0, the second value is 1, the third value is 2, etc, let's test this out:

JavaScript Code:

<Script> var colors = new Array("Red", "Brown", "Pink"); colors[2] = "Blue"; document.write( colors[1], " ", colors[2] ); </Script>

Outcome:

If you attempt to access an index outside of the array, it will return undefined:

JavaScript Code:

<Script> var colors = new Array("Red", "Brown", "Pink"); document.write( colors[10] ); </Script>

Outcome:

Other Ways to Create Arrays

You can also declare an array, tell the number of elements it will store and add the elements later, like this:

JavaScript Code:

<Script> var colors = new Array(3); colors[1] = "Red"; colors[2] = " Brown"; colors[3] = " Pink."; document.write( colors ); </script>

Outcome:

However, there really isn't a limitation to the number of values you can add, even tho you have wrote 3 if you were to add a fourth value it would still do it, so let's try the same thing, but not specify the number of values we are going to add:

JavaScript Code:

<Script> var colors = new Array(); colors[1] = "Red"; colors[2] = " Brown"; colors[3] = " Pink."; document.write( colors ); </script>

Outcome:

Now, I still am not sure why, but from what I read using new Array() is not recommended, for greater simplicity, readability and execution speed you can declare arrays using the array literal syntax, like this:

JavaScript Code:

<Script> var colors = ["Brown", "Red", "Pink"]; document.write( colors[0], " ", colors[2] ); </script>

Outcome:

Array Properties and Methods

JavaScript arrays have useful built-in properties and methods, an array's length property for example returns the number of it's elements.

JavaScript Code:

<script> var colors = ["Red", "Pink", "Brown"]; document.write( colors.length ); </script>

Outcome:

JavaScript's contact() method allows you to join arrys and create an entirely new array, like in the example below:

JavaScript Code:

<script> var colors1 = ["Brown", "Red"], colors2 = ["Green", "Blue"]; var all_colors = colors1.concat(colors2); document.write( all_colors ); </script>

Outcome:

You can also contact only some of the elements of choice by specifying them inside the array, using [].

JavaScript Code:

<script> var colors1 = ["Brown", "Red"], colors2 = ["Green", "Blue"]; var all_colors = colors1.concat(colors2[0]); document.write( all_colors ); </script>

Outcome:

Associative Arrays

While many programming languages support arrays with named indexes (text instead of numbers), the so called associative arrays, Javascript does not. However, there is a way to do something similar to associative arrays, by using the following syntax:

JavaScript Code:

<script> var person = []; person["name"] = "John"; person["age"] = 45; document.write(person["age"]); </script>

It's not really recommended to use this method tho, use objects instead.

The setInterval() method

The setInterval() method calls a function or evaluates an expression at specified intervals (in miliseconds). It will continue calling the function until clearInterval() is called or the window is closed.

JavaScript Code:

<Script> function fu(){ alert("Hello World!"); }; function example8(){ this.start = function(){ alertInterval = setInterval( fu, 6000); }; this.stop = function(){ clearInterval(alertInterval); }; }; var alerts = new example8(); </Script>

HTML Code:

<button onclick="alerts.start()" class="jsbutton">Start</button> <br><br> <button onclick="alerts.stop()" class="jsbutton">Stop</button>

Outcome:



What I created is a simple object constructor that contains two methods, the first method is called start and it basically makes the setInterval activate which repeats the fu fuction every three seconds, I also mentioned that this will be equal to alertInterval which doesn't mean anything, it's just a name I gave it so I can referense it later inside the stop method, which clears the alertInterval therefore stopping the loop. The Date Object

The date object enables us to work with dates, a date consists of a year, a month, a day, an hour, a minute and milliseconds.

By using the new Date() you can create a new date object with the current date and time, isn't that cool?

JavaScript Code:

<script> var today = new Date(); document.write( today ); </script>

Outcome:

The other ways to initialize dates allow for the creation of new date objects from the specified date and time.

<script> //Fri Jan 02 1970 00:00:00 var d1 = new Date(86400000); //Fri Jan 02 2015 10:42:00 var d2 = new Date("January 2, 2015 10:42:00"); //Sat Jun 11 1988 11:42:00 var d3 = new Date(88,5,11,11,42,0,0); var br = "<br>"; document.write( d1, br, d2, br, d3); </script>

Outcome:

When a date object is created a number of methods can be used to perform operations on it, here is a list:

JavaScript Code:

<script> var date = new Date(); document.write( date.getHours() ); </script>

Outcome:

Now that we know how to use methods let's make a real life time clock.

Code

<script> function clock(){ var d = new Date(); document.body.innerHTML = d.getHours() + ":" + d.getMinutes() + ":" + d.getSeconds(); setInterval( clock, 1000 ); }; </script> <button class="jsbutton" onclick="clock()">Click</button>

Outcome

If you now let our little clock work for a while you will notice that when a number gets under 10 it will write 1, 2, etc, instead of 01, 02, 03, like in a real clock, lets create a function which will check if the number is under 10 and fix this problem using simple if else statement.

Code

<script> function addZero(x){ if ( x < 10 ) { return "0" + x } else { return x }; } function clock2(){ var d = new Date(); var hour = addZero( d.getHours() ); var min = addZero( d.getMinutes() ); var sec = addZero( d.getSeconds() ) ; document.body.innerHTML = hour + ":" + min + ":" + sec; setInterval( clock2, 1000); }; </script> <button class="jsbutton" onclick="clock2()">Click</button>
