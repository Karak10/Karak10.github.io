JavaScript Functions

Just like variables, we also have Functions in JavaScript, which are very useful, since they allow you to reuse code very easily, and use it as many times as needed, without having to write everything over and over again.

See code below for example:

JavaScript Code:

<script> function hello(){ document.write("Hello World <br>"); } hello(); hello(); hello(); </script>

Outcome:

A function name can contain letters, digits, underscores and dollar signs, it basically has the same rules I wrote about naming variables before.

Functions can take parameters, function parameters are names listed in the function's definition, see the code below for example:

JavaScript Code:

<script> function hello(greeting, name){ document.write(greeting + name, "I'm Karak10 and I welcome you. <br>") } hello("Hello, ", "Jack "); hello("Hi, ", "Okabe "); </script>

Outcome:

Here is one more example to help you understand how it works:

JavaScript Code:

<script> function example(x, y){ if ( x > y ){ document.write( x, "<br>"); } else { document.write( y, "<br>" ); } } example(6, 5); example(5, 6); </script>

Outcome:

Here is another cool example I made, check this out:

JavaScript Code:

<Script> function example(money, price){ var money_needed = "In order to buy this product you will need "; var money_left = "If you buy this product you will be left with " if (money <= price){ document.write(money_left, money - price, "$"); } else { document.write(money_needed, price - money, "$"); } } example(30, 60); </Script>

The return Statement

A function can have an optional return statement. It is used to return a value from the function, this return statement is very useful when making calculations that require a result.

JavaScript Code:

<Script> function example(x, y){ var string1 = "If you buy this product you will be left with "; var string2 = "In order to buy this product you will need "; if (x > y){ return string1 + (x - y) + "$"; } else { return string2 + (y - x) + "$"; } } var result = example(10, 5); document.write(result); </Script>

Outcome:

Introduction to JavaScript Objects

JavaScript variables are containers for data values. Objects are variables too, but they can contain multiple values.

Think of an object as a list of values that are written as name:value pairs, with the names and values seperated by colons.

See the code below for example:

JavaScript Code:

<Script> var person = { name: "Alex", age: "15", sex: "male" }; document.write("My name is ", person.name, ", I'm a ", person.sex, " and I'm ", person.age, "years old." ); </Script>

Outcome:

In the example I accessed object properties by simply writing objectName.propertyName, there is however another way to do that too, by writing objectName['propertyName'].

An object method is a property that contains a function definition, to access an object method you should write objectName.methodName(), as you already know, document.write() outputs data, the write() is actually a method of the document which is the object.

JavaScript Code:

<Script> var page = { write: function(x){ document.write(x); } } page.write("Hello World!"); </Script>

Outcome:

JavaScript's build-in length property is used to count the number of characters in a property or string, empty spaces also count as characters.

JavaScript Code:

<Script> var name = "Alex"; document.write(name.length); </Script>

Outcome:

Another build-in property is the substring() property, which we can use to select specific parts of a string instead of selecting the whole string, see the example below:

JavaScript Code:

<Script> var alpha = "ABCD"; document.write( alpha.substring(1, 4) ); </Script>

Outcome:

It selects everything after first character and before fourth character, so that's why we get BCD as the result.

The Object Constructor

In the section above, I talked about how we can create objects using the object syntax, like this:

JavaScript Code:

<script> var person = { name: "Alex", age: 15 }; document.write("My name is ", person.name, " and I'm ", person.age, " years old."); </script>

Outcome:

This however allows you to create only one object, for example, if I wanted to have a person2 and give him the same properties, I would have to make a new object, which takes time and space in your code, worry not however, for I, the great Karak have a solution for you, which solution allows you to create object types using object constructor function.

JavaScript Code:

<script> function human(n, a){ this.name = n; this.age = a; } var p1 = new human("Alex", 15, "Male"); var p2 = new human("Christina", 18, "Female") document.write( p1.name, " is ", p1.age, "<br>" ); document.write( p2.name, " is ", p2.age ); </script>

Outcome:

Alright, so, let me do the explaining now, human is an object type, or else our object constructor, it contains properties named name and age, to add properties you must first use the this keyword, between this and the name of your property you should add a dot, we have assinged the name property of our human object constructor to be the same with the first function paremeter, which is n, I talked about how function paremeters work before, we did the same for age.

Now, by using the new keyword we are able to create new objects, of the human object type, which we call p1 and p2, and by taking advantage of the function parameters we simply write what we want the values for each object to be, without having to rewrite the properties again and again, we only assign the values, it is a little confusing, play around with it to learn it better.

Simplified conclusion: the new keyword creates a new object, then, the this keyword that is used inside the function assigns the properties and values of the object.

Adding Methods

Methods are functions that are stored as object properties. A method is a function, belonging to an object. I have already used methods in simple objects before, let's now try to add a method in an object constructor, this method I will create will allow you to change the values of an object by simply using .change() method, a method I created, see the code below:

JavaScript Code:

<script> function human(n, a){ this.name = n; this.age = a; this.change = function(n2, a2){ this.name = n2; this.age = a2; }; }; var p = new human("Alex", 15); p.change("Christina", 18); document.write( p.name, " ", p.age ); </script>

Outcome:
