jQuery Overview

In this section we will learn some basics about what jQuery is and how you can start using it, it's basically a preparation section for the following lessons.

What is jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, and animation much simpler.

All the power of jQuery is accessed via JavaScript, so having a strong grasp of JavaScript is essential for understanding, structuring, and debugging your code.

First, let's take a look at an example HTML manipulation with JavaScript. To get the element with the id="start" and change its html to "Go" we will need to do the following:

var el = document.getElementById("start"); el.innerHTML = "Go";

To do the same manipulation with jQuery, we need just a single line of code:

$("#start").html("Go");

We will learn about the new syntax in the coming lessons, but as you can see, the code is much shorter and easier to understand.

Basics about jQuery Syntax

jQuery is used to select (query) HTML elements and perform "actions" on them.

Basic syntax is: $("selector").action()

  • The $ accesses jQuery
  • The (selector) finds HTML elements
  • The action() is then performed on the element(s).

For example:

$("p").hide() // hides all <p> elements $(".demo").hide() // hides all elements with class="demo" $("#demo").hide() // hides the element with id="demo"

It is a good practice to wait for the HTML document to be fully loaded and ready before working with it. For that we use the ready event of the document object:

$(document).ready(function() { // jQuery code goes here });

Since the code above is used in almost all cases when using jQuery, there is a handy shortcut for writing it:

$(function() { // jQuery code goes here });

Attributes and Content

In this sections we will talk about how you can manipulate the attributes and content of an element using jQuery.

How to get, set or remove attributes

We can manipulate attributes assigned to HTML elements easily through jQuery. href, src, id, class, style are all examples of HTML attributes. The attr() method is used for getting the value of an attribute, for example:

<a href="www.sololearn.com"> Click here </a>

$(function() { var val = $("a").attr("href"); alert(val); });

The attr() method also allows us to set a value for an attribute by specifying it as the second parameter.

$(function() { $("a").attr("href", "http://www.jquery.com"); });

You can also remove attributes from HTML elements. The removeAttr() method is used for removing any attribute of an element. In the example below we remove the border and class attributes of the table:

$(function() { $("table").removeAttr("border"); $("table").removeAttr("class"); });

Get & Set Content

There are several methods for manipulating the content of HTML elements via jQuery. The html() method is used to get the content of the selected element, including the HTML markup, for example:

<p> JQuery is <b>fun</b> </p>

$(function() { var val = $("p").html(); alert(val); }); // alerts "JQuery is <b>fun</b>"

Notice, that the HTML markup (the <b> tags) is also returned, If you need only the text content, without the HTML markup, you can use the text() method:

$(function() { var val = $("p").text(); alert(val); }); // alerts "JQuery is fun"

The same html() and text() methods can be used to change the content of HTML elements. The content to be set is provided as a parameter to the method, for example:

<div id="test"> <p>some text</p> </div>

$(function() { $("#test").text("hello!"); console.log($("#test").html()) });

So, by running the code above you will notice two things, the text has indeed changed from "some text" to "hello!", and the string that is returned by the console.log statement also is "hello!", so the text() method has changed the content of the div, not just the text, the div has no paragraph inside it anymore. Using the html() method would do the same, however, if you wrote text('<b>hello!</b>') then that's where you would see a difference, the text method would change the text to "<b>hello!</b>", the html() method however would treat the <b> tags as HTML elements and not as text, so the text would become a bold "hello!". In other words, the html() method changes the HTML content inside an element, while the text() method changes the text conent inside an element.

The val() Method

We have seen in the previous lesson how we can manipulate the content of HTML elements using the text() and html() methods. Another useful method is the val() method, which allows us to get and set the values of form fields, such as textboxes, dropdowns, and similar inputs. For Example:

<p> <input type="text" id="name" value="Your Name"> </p>

$(function() { alert($("#name").val()); });

So, you may ask now, why not use the attr() method? Using attr('value') would always return "Your Name", meanwhile val() returns the actual written value that could be changed by the user.

Similarly, you can set the value for the field by providing it as a parameter to the val() method.

$(function() { $("#name").val('Karak10 is cool') });

Adding Content

As we have seen in the previous lessons, the html() and text() methods can be used to get and set the content of a selected element. However, when these methods are used to set content, the existing content is lost. jQuery has methods that are used to add new content to a selected element without deleting the existing content:

  • append() inserts content at the end of the selected elements.
  • prepend() inserts content at the beginning of the selected elements.
  • after() inserts content after the selected elements.
  • before() inserts content before the selected elements.

The append() method inserts content AT THE END of the selected HTML element. For example: HTML:

<p id="demo">Hi </p>

$(function() { $("#demo").prepend("<b>"); $("#demo").append("David<b>"); }); // Outputs: <p id="demo"><b>Hi David</b></p>

The jQuery after() and before() methods insert content AFTER and BEFORE the selected HTML element, and those methods can also use HTML markup for the content, for example:

$(function() { $("#demo").before("<h2>Some Title</h2>"); $("#demo").after("<p>Welcome</p>"); }); // Outputs: // <h2>Some Title</h2> // <p id="demo">Hi </p> // <p>Welcome</p>

The append(), prepend(), before() and after() methods can also be used to add newly created elements. The easiest way of creating a new HTML element with jQuery is the following:

var txt = $("<p></p>").text("Hi");

The code above creates a new <p> element, which contains the text Hi and assigns it to a variable called txt. Now, we can use that variable as a parameter of the above mentioned methods to add it to our HTML, for example

$(function() { var txt = $("

").text("Hi"); $("#demo").after(txt); });

This will insert the newly created <p> element after the #demo paragraph. You can also specify multiple elements as arguments for the before(), after(), append(), prepend() methods by separating them using commas: $("#demo").append(var1, var2, var3).

Manipulating the CSS

In this section you will learn how you can edit the CSS styles of elements using jQuery, as well as how you can add or remove classes:

Adding & Removing Classes

jQuery has several methods for CSS manipulation. The addClass( ) method adds one or more classes to the selected elements, for example:

<div>Some text</div>

.header { font-size:x-large; } .blue { color: blue; }

$('div').addClass("header blue");

The removeClass() method removes one or more class names from the selected elements.

$('div').removeClass("header blue");

The toggleClass() method toggles between adding/removing classes from the selected elements, meaning that if the specified class exists for the element, it is removed, and if it does not exist, it is added. To demonstrate this in action, we will handle a button click event to toggle a class. We will learn more about events and their syntax in the coming lessons.

<p>Some text</p> <button>Toggle Class</button>

.red { color:red; font-weight: bold; }

$(function() { $("button").click(function() { $("p").toggleClass("red"); }); });

The code above toggles the class name "red" upon clicking the button.

CSS Properties

Similar to the html() method, the css() method can be used to get and set CSS property values, for example:

<p>Some text</p>

p { background-color:red; color: white; }

$(function() { $("p").css("background-color", "blue"); alert($("p").css("background-color")); }); // this will alert "blue"

To set multiple CSS properties, the css() method uses JSON syntax, which is:

css({ "property1": "value1", "property2": "value2", "property3" : "value3" });

As you can see, the syntax consists of "property":"value" pairs, which are comma separated and enclosed in curly brackets { }.

Dimensions

The width() and height() methods can be used to get and set the width and height of HTML elements.

Let's set both the width and height of a div to 100px, as well as set a background color for it:

$(function() { $("div").css("background-color", "red"); $("div").width(100); $("div").height(100); }); // Note: the sizes in default are pixels

You can also do all these settings in one statement like this :

$(function() { $("div").css("background-color", "red").width(100).height(100); });

The width() and height() methods get and set the dimensions without the padding, borders and margins.

The innerWidth() and innerHeight() methods also include the padding. The outerWidth() and outerHeight() methods include the padding and borders.

The following example demonstrates how the methods work:

<div></div>

div { width: 300px; height: 100px; padding: 10px; margin: 20px; border: 3px solid blue; background-color: red; color: white; }

$(function() { console.log($('div').width()) console.log($('div').innerWidth()) console.log($('div').outerWidth()) }); // outcome: 300, 320, 326

width() is equal to 'width' in CSS when 'box-sizing: content-box' outerWidth() is equal to 'width' in CSS when 'box-sizing: border-box'

Manipulate the DOM

In this section we will learn how to manipulate the DOM using jQuery, however, I will not go over the basics of what the DOM is, what a parent element is, what a child element is, what a sibling element is, and so on, and that's because I'm pretty sure i have already explained those stuff in previous notes, but you can just google this stuff too if you want there really aren't that much you need to know, you already know HTML, so you probably already have a basic idea of hoe the DOM is structured and stuff.

DOM Traversal

jQuery has many useful methods for DOM traversal. The parent() method returns the direct parent element of the selected element. For example:

<div> div element <p>paragraph

</div>

$(function() { var e = $("p").parent(); e.css("border", "2px solid red"); });

The code above selects the parent element of the paragraph and sets a red border for it.

The parent() method can only traverse a single level up the DOM tree. To get all ancestors of the selected element you can use the parents() method. For example:

<body> body <div style="width:300px;"> div <ul> ul <li> li <p>paragraph</p> </li> </ul> </div> </body>

$(function() { var e = $("p").parents(); e.css("border", "2px solid red"); });

The code above sets a red border for all parents of the paragraph.

Some of the most used traversal methods are presented below:

Method What It Returns
parent() direct parent element of the selected element
parents() all ancestor elements of the selected element
children() all direct children of the selected element
siblings() all sibling elements
next() / nextAll() next or all next sibling elements of the selected element
prev() / prev previous or all previous sibling elements of the selected element
eq() element with a specific index number of the selected elements

You may be confused about the eq() method, do not be, it's simple, let's say you wanted to select the first paragraph in your document, if you wrote $('p') that would return a node array of all your paragraphs, to select the first one you could write $('p').eq(0), to select the second you can change 0 to 1, the index works just like an array index.

Removing Elements

We remove selected elements from the DOM using the remove() method, for example:

<p style="color:red">Red</p> <p style="color:green">Green</p> <p style="color:blue">Blue</p>

$(function(){ $('p').eq(1).remove(); });

This removes Green, the second paragraph element. You can also use the remove() method on multiple selected elements, for example $("p").remove() removes all paragraphs. The jQuery remove() method removes the selected element(s), as well as its child elements.

The empty() method is used to remove the child elements of the selected element(s), for example:

<div> <p style="color:red">Red</p> <p style="color:green">Green</p> <p style="color:blue">Blue</p> </div>

$(function(){ $("div").empty(); });

This removes all the three child elements of the div, leaving it empty.

Events

In this section we will talk about how you can use events with jQuery.

Handling Events

JQuery provides an efficient way to handle events. Events occur when the user performs an action, such as clicking an element, moving the mouse, or submitting a form. When an event occurs on a target element, a handler function is executed.

For example, let's say we want to handle the click event on an element with id="demo" and display the current date when the button is clicked. Using pure JavaScript, the code looks like:

window.addEventListener('load', function(e)=>{ var el = document.getElementById('demo'); el.addEventListener('click', function(e)=>{ document.body.innerHTML = Date(); }); });

The same event could be handled using jQuery with the following code:

$(function() { $('#demo').click(function(e) { $('body').html(Date()); }); });

As you can see, the jQuery code is shorter and easier to read and write. Notice, that the event name is provided without the "on" prefix.

Common Events:

Mouse Events:

click - occurs when an element is clicked.
dblclick - occurs when an element is double-clicked
mouseenter - occurs when the mouse pointer gets over (enters) the selected element.
mouseleave - occurs when the mouse pointer leaves the selected element
mouseover - occurs when the mouse pointer is over the selected element
mouseout - occurs when the mouse pointer is no longer over the selected element
mousemove - occurs when the mouse is over the selected element and also is moving

You can see the difference between mouseenter / mouseleave and mouseover / mouseout events when your target element contains child elements: Each time your mouse enters or leaves a child element, mouseover or mouseout is triggered, but not mouseenter or mouseleave. The mouseenter event only triggers when the mouse pointer enters the selected element and the mouseleave only triggers when the mouse leaves the selected element.

Keyboard Events:

keydown - occurs when a keyboard key is pressed down.
keyup - occurs when a keyboard key is released.

Form Events:

submit - occurs when a form is submitted.
change - occurs when the value of an element has been changed.
focus - occurs when an element gets focus.
blur - occurs when an element loses focus.

Document Events:

ready - occurs when the DOM has been leaded.
resize - occurs when the browser window changes size.
scroll - occurs when the user scrolls in the specified element.

Another way to handle events in jQuery is by using the on() method. The on() method is used to attach an event to the selected element. For example:

$("P").on("click", function(){ alert("Clicked"); });

The on() method is useful for binding the same handler function to multiple events. You can provide multiple event names separated by spaces as the first argument. For example, you could use the same event handler for the click and dblclick events like this: $('p').on('click dblclick', function(){}).

You can remove event handlers using the off() method:

$('div').on('click', function(){ alert('Hi there!'); $('div').off('click'); });

The Event Object

Every event handling function can receive an event object, which contains properties and methods related to the event:

  • pageX, pageY - the mouse position (X & Y coordinates) at the time the event occurred, relative to the top left of the page.
  • type - the type of the event (e.g. "click").
  • which - the button or key that was pressed.
  • data - any data that was passed in when the event was bound.
  • target - the DOM element that initiated the event.
  • preventDefault() - prevent the default action of the event (e.g., following a link).
  • stopPropagation() - Stop the event from bubbling up to other elements.

I have talked much more about events in my vanilla JavaScript notes so I will not explain them again.

As an example, letâ€™s handle the click event on an <a> element and prevent it from following the link provided in the href attribute:

$('a').click(function(event) { alert(event.pageX); event.preventDefault(); });

The code above alerts the mouse position at the time of the click and prevents following the link.

We can also trigger events programmatically using the trigger() method. For example, you can trigger a click event without the user actually clicking on an element:

$('div').click(function() { alert('Clicked!'); }); $('div').trigger('click');

The trigger() method cannot be used to mimic native browser events, such as clicking on a file input box or an anchor tag. Only events in the jQuery event system can be handled.

The animate() method

The animate() method lets you animate to a set value, or to a value relative to the current value. You need to define the CSS properties to be animated as its parameter in JSON format ("key":"value" pairs). The second parameter defines the speed of the animation. For example, the following code animates the width and height properties of the div:

<div id="special_div"></div>

#special_div { width: 100px; height: 100px; background: black; margin: auto; }

let div = $('#special_div') let x = 1; div.on('click', function(){ if(x == 1){ div.animate({ width: '250px', height: '50px' }, 1000); x = 0; } else { div.animate({ width: '100px', height: '100px' }, 1000); x = 1; } });

Outcome:

Note the JSON format for providing the CSS parameters. The JSON syntax was also used in the previous modules when manipulating CSS properties.

You can animate any CSS property using the above mentioned syntax, but there is one important thing to remember: all property names must be camel-cased when used with the animate() method (camelCase is the practice of writing compound words or phrases such that each word or abbreviation begins with a capital letter with the first word in lowercase). You will need to write paddingLeft instead of padding-left, marginRight instead of margin-right, and so on.

It is also possible to define relative values (the value is then relative to the element's current value). This is done by putting += or -= in front of the value:

let div2 = $('#cool_div'); div2.on('click', function(){ div2.animate({width: '+=100px'}, 1000) });

Outcome:

By default, jQuery comes with queue functionality for animations. This means that if you write multiple animate() calls one after another, jQuery creates an "internal" queue for these method calls. Then it runs the animate calls one-by-one.

The animate() method, just like the hide/show/fade/slide methods, can take an optional callback function as its parameter, which is executed after the current effect is finished.
