How to calculate Network, Broadcast and host addresses

So, I talked before about private IPs and subnet mask, but I wanted to talk a little more about this, since by knowing a subnet mask of an ip address you are able to tell what the network and braodcast IPs are, as well as how many hosts there can be in the netword..

So, for our example we will use this IP address:

172.16.5.34 /26

10101100.00010000.00000101.00100010

So, what is that /26 I put at the end? The /number you may see after ip addressed indicates the subnet mask of the network. To better understand what I mean let's take a look at the subnet mask of the ip above:

255.255.255.192

11111111.11111111.11111111.11000000

I want you to notice one thing about this subnet mask, the first 26 bits are all 1, while the rest 6 are 0. That's what the /26 means. The most common subnet mask you will see is usually 255.255.255.0, so that means you will usually see /24 at the end of IPs.

So now, what would the network address be? To find this out, we will use the IP I showed you before, and the subnet mask we talked about now, all the bits that are 1 in our subnet mask represent thet network portion of our IP address, while all the 0 bits represent the host portion of the IP address, knowing this, we can now find what the network address is, by simple ignoring the host portion bits, as if they were 0:

172.16.5.0

10101100.00010000.00000101.00000000

So, that's what we call the network address, our first host, which is usually our default gateway (the router) will have the first IP address, you can find this IP address by changing the last bit at the host portion of our IP adress to 1:

172.16.5.1

10101100.00010000.00000101.00000001

A broadcast address is a network address used to transmit to all devices connected to a multiple-access communications network. A message sent to a broadcast address may be received by all network-attached hosts. The broadcast address is the last IP address of our network, so in order to find what it is we should change all bits in host portion of our IP address to 1:

172.16.5.63

10101100.00010000.00000101.00111111

Calculating the number of hosts that our network can have is easy now, so, 172.16.5.0 is used as our network id, no host can use this as their IP address. 172.16.5.63 is used as our broadcast address, no host can have this IP address either. How many numbers are between 0 and 63? 62 numbers, so our network can have a max number of 62 hosts.
