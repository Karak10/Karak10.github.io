Introduction to HTML

Whats HTML?

HTML is a computer language devised to allow website creation. These websites can then be viewed by anyone else connected to the Internet.

The definition of HTML is HyperText Markup Language.

  • HyperText is the method by which you move around on the web — by clicking on special text called hyperlinks which bring you to the next page. The fact that it is hyper just means it is not linear — i.e. you can go to any place on the Internet whenever you want by clicking on links — there is no set order to do things in.
  • Markup is what HTML tags do to the text inside them. They mark it as a certain type of text (italicised text, for example).
  • HTML is a Language, as it has code-words and syntax like any other language.

HTML is just the "skeleton" of the page, the "skin" is CSS, the "muscles" is JavaScript, and after that there are other languages and tools that help us create a website like PHP language and others.

How to change line and how to change paragraph:

To change line you either close the paragraph by typing </p> and open another paragraph by typing <p> or break the line by typing <br>, when you break the line you continue writing right under the line, when you make another paragraph there is a bigger space between the lines because you actually change the paragraph.

Something I wanna add is the <pre> element, whatever is between <pre> and </pre> will be spaced like how it is spaced in the code, for example, if you write a paragraph and decide to change line, <pre> element will make the line break in your site too.

How to add images and comments.

About images:

To put an image you need to have the image in the same folder with your html document, then simply write the following code: <img src="../images/smile.jpg">

You can costumize the image with attributes such as width, height, border and others.

Border is the area around the picture, you can specify the size of the border by writing border="3px" or whatever other number you want, there are of course other ways to count as well, but I prefer using pixels.

To change the color of the border, without using CSS, you can use this code:

<font color="red">
<img src="../images/smile.jpg" height="50%" width="50%" border="3px" alt="smile">
</font>

Important: the <font> tag is not available in HTML5, <span> and <div> together with CSS should be used instead.

The alt="smile" attribute is what I want it to write if the image isn't supported by a browser, or if for some reason it doesn't load.

If you want to add a figcaption to your image you first type <figure>, then <img src="../images/smile.jpg" etc etc>, then to add a figcaption in your image you put your figcaption between <figcaption> and </figcaption>, you can put more figaptions than one if you want, then you close figure tag with </figure>, all this is put between <font> and </font> if you wanna add color to your image, the color will also be added to the figcaptions.

Code:

<font color="red">
<figure>
<img src="../images/smile.jpg" border="3px">
<figcaption> <cite>Smile</cite> </figcaption>
<figcaption>By Karak10 </figcaption>
</figure>
</font>

Outcome:

To change the position of the image you have to put it in a paragraph like this: <p align="center"> <img src="../images/smile.jpg" border="3px"> </p>

These are the target types you can use and what each does.

Values of target attribute:
Value Description
_blank Opens the linked document in a new window or tab example
_self Opens the linked document in the same frame as it was clicked (this is default) example
_parent Opens the linked document in the parent frame, the result is the same with _self value, the difference is only visible when used with iframe. example
_top Opens the linked document in the full body of the window, the result is the same with _self value, the difference is only visible when used with iframe. example

Another element used for links is the base element, this element ise inside the head element and only one base element can be used in a document. The base tag specifies the base URL and/or arget for all relative URLs in a document. The base element must have either an href or a target attribute present, or both.

For example, if I wrote <base target="_blank"> all hyperlinks and forms in the page would open in another pad when pressed. If all the hyperlinks I used were part of the same site, for example, my notes, I could write <base href="http://karak10notes.c1.biz/"> and then if I wanted to add another link that takes you to the second page of my notes for example I could simply write <a href="../HTML-Notes/Notes2.html">, without writing the whole URL, which is http://karak10notes.c1.biz/Notes2.html, I hope you understand.
