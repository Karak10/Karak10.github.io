Box-sizing Property

To understand box-sizing property you first need to understand the box model, as I have said in previous CSS lessons, the width of an element is calculated by the width, padding and border, because of that, usually you face problems where an element is overflowing it's parent, etc, and you need to be very careful and calculate the combined size of the border, padding and width every single time to make it exactly the wanted size.

Let's take a look to the following elements now:

HTML Code:

<div id="el1">First Element</div> <div id="el2">Second Element</div> <div id="el3">Third Element</div>

CSS Code:

#el1, #el2, #el3{ width: 500px; height: 20px; padding: 50px; border: 3px solid black; } #el1 { background-color: yellow; } #el2 { background-color: brown } #el3 { background-color: lightgreen }

Outcome:

First Element
Second Element
Third Element

If I wanted to change the padding of the first element and the border of the third element, the symetry of my creation would be totally ruined, without making the necessary changes calculated by some time consuming math:

CSS Code:

#el1 { background-color: yellow; padding: 20px; } #el3 { background-color: lightgreen; border: 10px solid black; }

Outcome:

First Element
Second Element
Third Element

With box-sizing property we can make element sizing much simplier. The default value of this property is content-box, which is the normal sizing you are already familiar with, if you change it to border-box however, something great happens, that makes element sizing a lot easier to handle:

CSS Code:

#e1el1, #e1el2, #e1el3{ width: 500px; height: 30px; padding: 50px; border: 3px solid black; box-sizing: border-box; }

Outcome:

First Element
Second Element
Third Element

Let me explain what happened, the width of the elements ramained to 500px, even though we changed the padding of the first element and the border of the third element. Instead of making the width bigger however, it shrank the content or made the content bigger to fill all the 500px. It takes border and padding and then adds the width necessary so that all together take 500px.

Usually you will see this property being used like in the code below:

CSS Code:

html{ box-sizing: border-box; } *::before, *::after{ box-sizing: inherit; }

That's to make sure all the elements are using border-sizing: border-box, even pseudo elements, this simple two lines of code make it easier for you to style your page without having to do a lot of math to create the project you want.

CSS Variables

Let's talk about CSS variables now, they were introduced in CSS3, and can help you a lot while writing style sheets. Usually, CSS variables are declared in either the root or the html of your document, and that's because CSS variables just like all CSS properties, cascade down and can be inherited or overridden in lower down more specific elements.:root pseudo-class matches the <html> element, and is identical to the selector html, except that it's specificity is higher.

HTML Code:

<div class="ex11">Hello World!</div>

CSS Code:

:root{ --var-1: red; --var-2: white; --var-3: 3px solid black; --var-4: 50px; } .ex11{ background-color: var(--var-1); color: var(--var-2); border: var(--var-3); width: var(--var-4); }

Outcome:

Hello World!

As you can see CSS variables store values that can later be used for properties, and that's cool, mainly becaue that means you can use a variable in multiple places and be able to change the value of all those places by changing the variable's value.

HTML Code:

<div class="D-one"> <div class="one-one child">1-1</div> <div class="one-two child">1-2</div> <div class="one-three child">1-3</div> </div> <div class="D-two"> <div class="two-one child">2-1</div> <div class="two-two child">2-2</div> <div class="two-three child">2-3</div> </div>

CSS Code:

:root{ --div-background-color: red; --div-color: black; } .child{ background-color: var(--div-background-color); color: var(--div-color); width: 100px; border: 1px solid black; } .D-one{ --div-background-color: blue; --div-color: white; }

Outcome:

1-1
1-2
1-3
2-1
2-2
2-3

As you can see, variables can be overriden if you redeclare them in lower down more specific elements

If you use an undeclared variable you can specify a fallback value to be used if the variable can't be found, that's useful when used together with JavaScript, since you can use JavaScript to create that CSS variable later:

HTML Code:

<div class="idkk">Hello World!</div>

CSS Code:

.idkk{ background-color: var(--background-color, pink); width: 100px; }

Outcome:

Hello World!

CSS Calc Function

In this section I'm gonna speak about the calc keyword that can be used in CSS to do calculations and I will explain why it is so useful. Firstly, let's try and set the width of a div using calc, a quite useless calculation indeed, but I will create more useful examples using this keyword later on the section, now I only create a simple calculation so that we can take a look to the syntax:

HTML Code:

<div id="calc1">Hello World!</div>

CSS Code:

#calc1{ background-color: orange; width: calc(25px + 25px); }

Outcome:

Hello World!

As you can see, the syntax is quite simple, it takes number of units, for example, pixels, percent, etc, and you can use mathematical operators to do a calculations, you need to be careful however, there have to be spaces between the numbers and the mathematical operators, writing 5px -10px wouldn't work, you have to write 5px - 10px. The calc functions can also be nested, like in the example below:

HTML Code:

<div id="calc2">Hello World!</div>

CSS Code:

#calc2{ background-color: lightcoral; width: calc((25 + calc(5 + 5) + 15) * 1px); }

Outcome:

Hello World!

So, one of the reasons calc is so useful, is because you can actually combine together different units, for example, percent and pixels like in the example below:

HTML Code:

<div id="parent-calc"> <div id="calc3"></div> </div>

CSS Code:

#parent-calc{ width: 100px; height: 100px; background-color: yellow; display: grid; justify-items: center; align-items: center; } #calc3{ background-color: lightpink; width: calc(100% - 20px); height: calc(100% - 20px); }

Outcome:

When calc really becomes very useful though, is when it's used together with CSS variables. For example, let's say I have two divs and I want to second div to always be half the width of the first div, this could be done using CSS variables and calc keyword like below:

HTML Code:

<div id="calc4">Hello World!</div> <div id="calc5">Hello World!</div>

CSS Code:

:root{ --width: 100px; } #calc4{ width: var(--width); background-color: pink; }

Outcome:

Hello World!
Hello World!

Now, if you ever decide to change the width variable, the second div will always be half the width of the first div, this combined with some JavaScript could do amazing things, but even without JavaScript, it still can be really helpful

The :is pseudo-class

So, usually, if we wanted to select multiple targets we would do something like this:

HTML Code:

<div id="div-example"> <h1>I'm a header</h1> <p>Hello, this is a paragraph and I just want to fill this paragraph so it looks cool, well, here is a link that gets you nowhere: <a href="./CSS_Extra_1.html">nowhere.com</a>, so, if you click that link it will just reload the page, because why not, and, to make this paragraph a little better here is another link: <a href="CSS_Extra_1.html">imalink.html</a>, I guess that should be enough, let's get into styling now. </p> </div>

CSS Code:

#div-example{ width: 500px; margin: auto; border: 3px solid black; background-color: red; padding: 20px; } #div-example p{ text-align: justify; line-height: 25px; font-weight: bold; } #div-example h1{ border: 3px solid black; border-radius: 30%; width: fit-content; padding: 20px; margin: auto; background-color: orange; } #div-example h1, #div-example a{ color: purple; }

Outcome:

I'm a header

Hello, this is a paragraph and I just want to fill this paragraph so it looks cool, well, here is a link that gets you nowhere: nowhere.com, so, if you click that link it will just reload the page, because why not, and, to make this paragraph a little better here is another link: imalink.html, I guess that should be enough, let's get into styling now.

So, what I want you to pay attention at is the orange colored code, as you can see, I wrote #div-example h1, #div-example a{ color: purple; } to select both targets, which is fine, but has some problems. First of all, it takes space and time, except from that, if even one of the selectors is misspelled the whol styling block will be ignored.

Another, better way to select multiple targets is to use the :is() pseudo-class. If I wanted for example to select h1 and a tags inside the div with the id #div-example, just like in the example I made before, I could write #div-example :is(h1, a){ color: purple }:

Style:

#div-example2 :is(a, h1){ color: green; }

The :is() pseudo-class can also be used behind other selectors, and the selectors inside it can be more complicated selectors:

Outcome:

I'm a header

Hello, this is a paragraph and I just want to fill this paragraph so it looks cool, well, here is a link that gets you nowhere: nowhere.com, so, if you click that link it will just reload the page, because why not, and, to make this paragraph a little better here is another link: imalink.html, I guess that should be enough, let's get into styling now.

HTML Code:

<div id="div-example3"> <grass> <h3>Grass</h3> <p>grass is really cool, it is green cows love to eat it all the time. </p> </grass> <tree> <leafs> <h3>Leafs</h3> <p>the leafs is the part of the tree that is green, it is really cool and beautiful. </p> </leafs> <wood> <h3>Wood</h3> <p>wood is the brown part of the tree, wood can be used for many things, and a friend of mine called Steve loves to punch it all the time. </p> </wood> </tree> </div>

CSS Code:

#div-example3{ width: 500px; margin: auto; border: 3px solid black; border-radius: 50%; overflow: hidden; background-color: lightsalmon; padding-bottom: 20px; } #div-example3 :is(grass, tree leafs) h3{ background-color: lightgreen; } #div-example3 wood h3{ background-color: brown; } #div-example3 p{ width: 300px; margin: auto; }

If I were not using the :is pseudo-class, to make the example above, in order to style the wanted headings I would need to write the following code:

#div-example3 grass h3, #div-example3 tree leafs h3{ background-color: lightgreen; }

The :is() pseudo-class just makes things way faster, and except from that, if one of the selectors you put inside is misspelled, the style still works for the other selectors, which is another great reason to use the :s() pseudo-class.
