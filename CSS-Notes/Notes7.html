The font-family Property

The font-family property specifies the font of an element.

There are two types of font family names:

-font family: a specific font family (like Times New Roman or Arial)
-generic family: a group of font families with a similar look (like Serif or Monospace)

Here is an example of different font styles:

Generic family Font family
Serif Times New Roman
Georgia
Sans - serif Arial
Verdana
Monospace Curier New
Lucida Console

HTML code:

<p class="serif">
This is a paragraph shown in serif font.
</p>

<p class="sansserif">
This is a paragraph shown in sans-serif font.
</p>

<p class="monospace">
This is a paragraph shown in monospace font.
</p>

<p class="cursive">
This is a paragraph shown in cursive font.
</p>

<p class="fantasy">
This is a paragraph shown in fantasy font.
</p>

CSS code:

p.serif {
font-family: "Times New Roman", Times, serif;
}
p.sansserif {
font-family: Helvetica, Arial, sans-serif;
}
p.monospace {
font-family: "Cursier New", Courier, monospace;
}
p.cursive {
font-family: Florence, cursive;
}
p.fantasy {
font-family: Blippo, fantasy;
}

Outcome:

This is a paragraph shown in serif font.

This is a paragraph shown in sans-serif font.

This is a paragraph shown in monospace font.

This is a paragraph shown in cursive font.

This is a paragraph shown in fantasy font.

I used comma to add more than one value, the added values are alternatives. If Time New Roman is not available in browser it will pick Times, if also not available it will pick similar from generic family which is serif.

If the name of a font family is more than one word, it must be in quotation marks: "Times New Roman"

The font-size Property

The font-size property sets the size of a font. One way to se the size of fonts on the web is to use keywords, for example xx-small, small, medium, large, larger, etc.

HTML code:

<p class="small">

Paragraph text set to be small
</p>

<p class="medium">
Paragraph text set to be medium
</p>

<p class="large">
Paragraph text set to be large
</p>

<p class="xlarge">
Paragraph text set to be x-large
</p>

CSS code:

p.small {
font-size: small;
}
p.medium {
font-size: medium;
}
p.large {
font-size: large;
}
p.xlarge {
font-size: x-large;
}

Outcome:

Paragraph text set to be small

Paragraph text set to be medium

Paragraph text set to be large

Paragraph text set to be x-large

You can also use numerical values in pixels, ems and others to manipulate font size.

Setting the font size in pixel values (px) is a good choice when you need pixel accurancy, and it gives you full control of the text size.

The em size unit is another way to set the font size (em is relative size unit). It allows all major browsers to resize the text. If you haven't se the font size anywhere on the page, then it is the browser default size, which is 16px.

To calculate the em size, juse use the following formula: em = pixels/16

HTML code:

<h1 class="px">
This is a 20px heading
</h1>

<h1 class="em">
This is a 1.25em heading
</h1>

CSS code:

h1.px {
font-size: 20px;
}
h1.em {
font-size: 1.25em;
}

Outcome:

This is a 20px heading

This is a 1.25em heading

Both of the examples will produce the same result in the browser, because 20/16=1.25em

To finish with this section, I will add here some of the values you can use for font size categorized in groups:

Absolute size values:
font-size: xx-small;
font-size: x-smal;
font-size: small;
font-size: medium;
font-size: large;
font-size: x-large;
font-size: xx-large;

Relative size values:
font-size: smaller; (makes the element smaller than it's parent)
font-size: larger; (makes the element larger than it's parent)

Length values:
font-size: 12px;
font-size: 0.8em;

Percentage values:
font-size: 80% (makes the element get the 80% size of it's parent)

Global values:
font-size: inherit; (Makes the element get same size with it's parent)
font-size: initial; (Gives element it's default size)
font-size: unset; (Resets a property to it's inherited value if the property naturally inherits from it's parent)

The font-style Property

The font-style property is typically used to specify italic text.

HTML code:

<p class="italic">
This is a paragraph in italic style.
</p>

CSS code:

p.italic {
font-style: italic;
}

Outcome:

This is a paragraph in italic style.

The font-style property has three values: normal, italic, and oblique.

The normal value doesn't do anything, it can be used if you want to overrun an older styling rule I believe, and thats pretty much the reason it exists, oblique value is pretty similar to italic but less supported.

The font-weight Property

The font-weight controls the boldness or thickness of the text. The values can be set as normal (default size), bold, bolder, and lighter.

Lets play around with font-weight now, to understand it better, I will also change the font-family to sans-serif so the difference is more visible.

HTML code:

<p class="sans"> This is a font with a "normal" weight.</p>

<p clas="light sans"> This is a font with a "lighter" weight.</p>

<p class="bold sans">This is a font with a "bold" weight.</p>

<p class="bolder sans">This is a font with a "bolder" weight.</p>

CSS code:

<style>
.light {
font-weight: lighter;
}
.bold {
font-weight: bold;
}
.bolder {
font-weight: bolder;
}
.sans {
font-family: sans-serif;
}
</style>

Outcome:

This is a font with a "normal" weight.

This is a font with a "lighter" weight.

This is a font with a "bold" weight.

This is a font with a "bolder" weight.

You can also define the font-weight with a number from 100 (thin) to 900 (thick), according to how thick you want the text to be, 400 is the same as normal, 700 is the same as bold

The font-variant Property

The font-variant property allows you to convert your font to all small caps. The values can be set as normal, small-caps and inherit, as I explained in the font-size section, inherit gives the element same value it's parent has, so if for example the body of the HTML document was set to font-variant: small-caps the paragraph would also be small-caps.

HTML code:

<p class="variant-normal"> Paragraph font variant set to normal. </p>

<p class="variant-smallcaps"> Paragraph font variant set to small-caps. </p>

CSS code:

<style>
.variant-normal {
font-variant: normal;
}
.variant-smallcaps{
font-variant: small-caps;
}
</style>

Outcome:

Paragraph font variant set to normal.

Paragraph font variant set to small-caps.

The color Property

The color property specifies the color of the text.

One method of specifying the color of the text is using a color name: like red, blue, green, etc. Here is an example of changing the color of your font.

HTML code:

<p> <span class="color-green"> The text inside the span is green. </span> The text outside the span is black (by default).</p>

CSS code:

<style>
.color-green {
color: green;
background-color: unset;
}
</style>

Outcome:

The text inside the span is green. The text outside the span is black (by default).

I had to set the background-color to unset because I have set all spans to have background-color: green;, something stupid, but I did that because I use <span> a lot to mark words in the notes and it would take time to write a class every single time.

Another way of defining colors is using hexademical values and RGB.

Hexademical form is a pount sign (#) followed by at most, six hex values (0-F).

RBG defines the individual values of Red, Green, and Blue.

Code:

<p style="color: #0000FF">This is a blue paragraph </p>

<p style="color: rgb(0,0,900)">This is also a blue paragraph </p>

Outcome:

This is a blue paragraph

This is also a blue paragraph

Another way to define color is with rgba method, it works just like rgb but there is one more value, that defines the opacity, 0 is invisible, 1 is visible.

The text-align Property

The text-align property specifies the horizontal alignment of text in an element. By default, text on your website is aligned to the left, however, at times you may require a different alignment.

Text-align property values are the following: left, right, center, and justify.

HTML code:

<p class="left"> This paragraph is aligned to <Strong>left</Strong>. </p>

<p class="center"> This paragraph is aligned to <Strong>center</Strong>. </p>

<p class="right"> This paragraph is aligned to <Strong>right</Strong>. </p>

CSS code:

<Style>
.left {
text-align: left;
}
.center {
text-align: center;
}
.right {
text-align: right;
}
</Style>

Outcome:

This paragraph is aligned to left.

This paragraph is aligned to center.

This paragraph is aligned to right.

The justify value stretches all lines except the last line to fill the width of the screen (or the containing element). This means justify won't have any effects on one line of text.

This is a justified paragraph I made for an example using text-align: justify, I don't really have anything to say so lets just talk about the weather, its pretty hot where I live, for now at least, since it's summer, it does get cold at winter, but I don't really mind, I like cold weather more than hot weather. Lets now talk about food, my favorite one is pizza, its just great, if I could eat pizza every single night I would definitely do that, I also like hot chocolate, and hate coffee, I just don't like the taste of it.

This is a left aligned paragraph I made for an example using text-align: left, I don't really have anything to say so lets just talk about the weather, its pretty hot where I live, for now at least, since it's summer, it does get cold at winter, but I don't really mind, I like cold weather more than hot weather. Lets now talk about food, my favorite one is pizza, its just great, if I could eat pizza every single night I would definitely do that, I also like hot chocolate, and hate coffee, I just don't like the taste of it.

As you can see the width of the two first lines of the first paragraph are the same, because text-align: justify; strenched the second line to fit in.

The vertical-align Property

The vertical-align property sets an element's vertical alignment. Commonly used values are top, middle, and bottom.

The example below shows how to vertically align the text between a table.

HTML code:

<table>
<tr>
<td class="tdTop">Top</td>
<td class="tdMiddle">Middle</td>
<td class="tdBottom">Bottom</td>
</tr>
</table>

CSS code:

<Style>

#table2 {
border: 3px solid black;
border-collapse: collapse;
height: 200px;
width: 250px;
text-align: center;
margin-left: auto;
margin-right: auto;
background-color: white;
}
.tdTop {
vertical-align: top;
border: 3px solid black;
}
.tdMiddle {
vertical-align: middle;
border: 3px solid black;
}
.tdBottom {
vertical-align: bottom;
border: 3px solid black;
}

</Style>

Outcome:

Top Middle Bottom

The vertical-align property also takes the following values: baseline, sub, super, % and px (or pt, cm, etc).

This is a sub line text example.

This is a super line text example.

Vertical-align works on inline elements directly without any issue. eg:<dt>, <span>. But while using it with block elements we have to use display property. eg: <div>.

The display property is a bridge to use vertical-align on block elements by visualizing them as inline elements.

Lets try to vertically align a div by making it display as a table data inside a table:

HTML code:

<div id="exampletable">

</div>



</div>

The styling rules applied to main div and the marging-left/margin-right rules applied to exampletable div were added in order to horizontally center the square.

The text-decoration Property

The text-decoration property specifies how the text will be decorated.

Commonly used values are:

  • none - The default value, this defines a normal text

  • inherit - Inherits this property from it's parent element

  • overline - Draws a horizontal line below the text

  • underline - Draws a horizontal line below the text

  • line-through - Draws a horizontal line through the text (substitudes the HTML <s> tag)

HTML code:

<p class="none"> This is default style of text (none). </p>

<p class="inherit"> This text inherits the decoration of the parent. </p>

<p class="overline"> This is overlined text. </p>

<p class="underline"> This is underlined text. </p>

<p class="line-through"> This is lined-through text. </p>

CSS code:

<Style>

.none {
text-decoration: none;
}

.inherit {
text-decoration: inherit;
}

.overline {
text-decoration: overline;
}

.underline {
text-decoration: underline;
}

.line-through {
text-decoration: line-through;
}

</Style>

Outcome:

This is default style of text (none).

This text inherits the decoration of the parent.

This is overlined text.

This is underlined text.

This is lined-through text.

You can also combine decorations: <p style="text-decoration: underline overline line-through;"> I'm a paragraph </p>

This is an example of <span>text-indent</span> property.
First line of our text is indented to the right in 400px. Besides pixels you can also use other measurement units, like pt, cm, em, etc.
</p>

Outcome:

This is an example of text-indent property. First line of our text is indented to the right in 400px. Besides pixels you can also use other measurement units, like pt, cm, em, etc.

The text-shadow Property

The text-shadow property adds shadow to text. It takes four values: the first value defines the distance of the shadow in the x direction, the second value sets the distance in the y direction, the third value defines the blur of the shadow, and the fourth value sets the color.

HTML code:

<h1 id="shadow-example"> Text-shadow example </h1>

CSS code:

<Style>

#shadow-example {
color:blue;
text-align: center;
font-size: 50px;
text-shadow: 5px 2px 4px gray;
}

</Style>

Outcome:

Text-shadow example

To add multiple shadows you can add comma.

HTML code:

<h1 id="shadow-example2"> Text-shadow example using multiple shadows </h1>

CSS code:

<Style>

#shadow-example2 {
color:blue;
text-align: center;
font-size: 50px;
text-shadow: 5px 2px 4px gray, 10px 4px 4px red, 15px 6px 4px purple, 20px 8px 4px green;
}

</Style>

Outcome:

Text-shadow example using multiple shadows

You can of course use other color formats too like rbg, rgba, hexademical, etc. You can also use other units for the x, y and blur paremeters like (px, cm, mm, in, pc, pt, etc)

The white-space Property

The white-space property specifies how white-space inside an element is handled. The values can be set as normal, inherit, nowrap, etc.

The nowrap value makes the text continue on the same line until a <br> tag is encountered, and also collapses all sequences of whitespace into a single whitespace.

Code:

<p style="white-space: nowrap;"> This is a paragraph and I now have to write some stuff so it gets big so its easier to understand how the nowrap works, lets make a big whitespace here( )which doesn't really matter because it will be collapsed into a single whitespace thanks to nowrap, so the whitespace will only be visible on code, also, all the paragraph will be written in a single line because no br tag is met.</p>

Outcome:

This is a paragraph and I now have to write some stuff so it gets big so its easier to understand how the nowrap works, lets make a big whitespace here( )which doesn't really matter because it will be collapsed into a single whitespace thanks to nowrap, so the whitespace will only be visible on code, also, all the paragraph will be written in a single line because no br tag is met.

The white-space property also supports other values:

pre - text will only wrap on line breaks and white spaces are visible

pre-line - text will wrap where there is a break in code, but extra white spaces are still ignored

pre-wrap - text will wrap when necessary (eg. when the line reaches the end), will wrap on line breaks and white spaces are visible.
